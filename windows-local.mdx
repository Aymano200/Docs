## Getting started

Learn how to host the site locally and and deploy it to the public.

Please ensure you have all prerequisites installed. This tutorial is for Windows users.
      <Accordion icon="rectangle-terminal" title="Installing Node">
Visit https://nodejs.org/en and download the reccomended version. 

Open the installer and complete the installation.
</Accordion>

<Steps titleSize="h3">
<Step title="Comment corriger les résultats de Google qui sont buggés ?">
Allez dans les paramètres d'Interstellar et changez votre navigateur par défaut pour Bing par exemple
</Step>
<Step title="Comment se connecter à GeForce NOW ?">
Voir ["Accéder à GeForce NOW"](/geforcenow) pour des instructions détaillées étape par étape
</Step>

<Step title="Certains jeux ne fonctionnent pas pour moi">
Essayez de changer le proxy dans les paramètres, cela pourrait fonctionner pour certains jeux spécifiques
</Step>

<Step title="Comment vider mon cache ?">
Pour les utilisateurs de Safari sur iPad, [voir ce guide](https://support.apple.com/en-ca/105082). Pour les utilisateurs de Microsoft Edge, [voir ce guide](https://www.microsoft.com/en[..])
</Step>
</Steps>

Pourquoi : Parfois, des choses se cassent sur Interstellar, vider votre cache peut résoudre certains (mais pas tous) problèmes courants.

Ce tutoriel vous montrera comment accéder à votre compte GeForce NOW via Interstellar.
---
 <Note>Cela fonctionne uniquement sur PC</Note>
---
1. Ouvrez GeForce NOW.
2. Connectez-vous.
3. Fermez l'onglet de connexion.
4. Rechargez l'onglet d'origine.
5. Trouvez un jeu et cliquez sur `Jouer`.
